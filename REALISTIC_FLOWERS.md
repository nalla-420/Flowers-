## 🌹 **The Rose**
*Symbol of Love & Romance*

### Visual Characteristics
- **7-Layer Structure**: Each layer contains progressively more petals, mimicking real rose botanical anatomy
- **Organic Layering**: Layers rotate at unique angles for natural, asymmetrical beauty
- **Curved Petals**: Bezier curves create realistic organic petal shapes with natural edge rolls
- **Fine Veining**: Ultra-thin white veins radiate from petal base to tip, capturing light naturally
- **Color Gradations**: Multi-stop gradients transition from primary → secondary → primary → translucent white
- **Natural Offsets**: Petals have random Y-offsets to avoid uniform, artificial symmetry
- **Central Carpel**: Deep gradient carpel with radiating stamens and golden pollen

### Bloom Sequence
1. Inner petals unfold first (tighter, more organized)
2. Outer petals gradually open (looser, more romantic)
3. Center stamens become visible as bloom progresses
4. Final translucent tip highlights emerge

### Real Inspiration
- Based on hybrid tea rose structure (David Austin varieties)
- Each petal exhibits natural wave and curl typical of heirloom roses
- Center detail matches Rosa damascena botanical anatomy

## 📊 **Performance Optimization**

### Canvas Rendering Efficiency
- **Lazy Computation**: Veins drawn only when bloom > 0.3
- **Gradient Caching**: Multi-stop gradients computed once per frame
- **Bezier Path Optimization**: Minimal curve resolution while maintaining quality
- **Opacity Layering**: Efficient transparency without blending overhead
- **Transform Stacking**: Rotate/translate batched for GPU acceleration

### Memory Management
- **One-Shot Path Creation**: Petals paths not stored, recreated each frame
- **Shared Color Palette**: 5 global color schemes reduce allocations
- **Procedural Generation**: Zero texture uploads, pure vector rendering
